> **Installation Methods :**

| Method    | Command                                                                                           |
|:----------|:--------------------------------------------------------------------------------------------------|
| **git**   | `git clone https://github.com/imegeek/Theme-Engine ; cd Theme-Engine ; ./theme-engine`                  |       
| **curl**  | `bash <(curl -fsSL "https://git.io/theme-engine")` |
| **wget**  | `bash -c "$(wget -O- https://git.io/theme-engine)"`   |
